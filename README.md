# Nikhil Jesani — Portfolio

A high-performance, visually stunning portfolio for **Nikhil Jesani**, an AI Automation Engineer and Agentic AI Builder. This portfolio showcases projects, skills, a professional journey, and provides a direct terminal-style contact interface.

## Tech Stack

This project is built with modern web technologies focusing on performance and fluid animations:
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **3D Rendering:** [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) & [Three.js](https://threejs.org/)
- **Scrolling:** [Lenis](https://lenis.studiofreight.com/) for buttery-smooth scrolling
- **Icons:** [Lucide React](https://lucide.dev/)

## Features

- **Immersive 3D Backgrounds:** A dynamic, interactive neural network background built with WebGL and React Three Fiber.
- **Agentic AI Branding:** Custom sections dedicated to highlighting AI multi-agent workflows, skills, and specific achievements.
- **Responsive Layouts:** Carefully crafted mobile and desktop views, ensuring pixel-perfect alignment.
- **Terminal Contact Form:** A playful and interactive developer-centric contact section.
- **Smooth Scrolling & Micro-interactions:** Every button and scroll event feels premium, thanks to Lenis and Framer Motion.

## Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd nikhil-jesani-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Structure

- `src/app`: Next.js App Router entry points, global layouts, and global CSS.
- `src/components/sections`: Page sections (Hero, About, Projects, Contact, Skills, etc.).
- `src/components/layout`: Core layout components like the Navbar and Footer.
- `src/components/three`: Three.js and React Three Fiber components for the 3D backgrounds.
- `src/components/ui`: Reusable UI elements (Custom Cursor, Section Headings).
- `public`: Static assets including images and the downloadable resume PDF.

## Deployment

This Next.js application can be seamlessly deployed on [Vercel](https://vercel.com/), the creators of Next.js. Simply import your GitHub repository into Vercel, and it will automatically build and deploy.
